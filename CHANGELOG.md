# Changelog

## [1.0.0] - 2025-11-29
### Added
- Initial release of Recipe Organizer.
- Recursive file search for `.docx`, `.doc`, `.pdf`, `.txt`, `.tiff`.
- Zip file support: Extracts recipes found inside archives.
- Keyword detection: "Ingredients", "Directions", "Method", "Yield", etc.
- Filename fallback detection.
- GUI Launcher (`RecipeOrganizer.exe`).
- PowerShell script with `Test`, `Copy`, and `Move` modes.
- `-NoRecurse` parameter for top-level only searches.
